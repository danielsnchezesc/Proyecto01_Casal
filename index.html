Proyecto 01 - Reserva de recursos

El objetivo principal para este proyecto era empezar una pagina web desde cero, se trata de una pagina web que funcionara como una intranet, donde los trabajadores de una entidad/casal puedan ver la disponibilidad de los recursos y poder reservarlos. Los recursos podrán estar libre o ocupados, una vez el recurso se haya devuelto, se tendrá que acceder a la pagina para poder desbloquearlo.

Tendrá que haber un historial sobre que día y hora se reserva/libera el recurso, aparte de esto, cada usuario tendrá que hacer logIn y logOut. A parte de esta intranet necesitaremos una base de datos donde guardemos toda esta información por lo tanto crearemos una base de datos también desde cero.

• Planificacion

Durante el proyecto hemos repartido las tareas que habían de manera equitativa, para así no sobrecargar de faena a ningún integrante del grupo mas que a otro.

Para ayudarnos a planificar y organizar el trabajo de el grupo en general hemos utilizado dos herramientas:

GitHUB

En github hemos utilizado el apartado de proyectos para organizar todas las issues/tareas para ir organizandolo todo de manera grupal, a medida que vamos avanzando en el proyecto.

Excel

Con excel hemos planificado de manera mas exacta, hemos especificado horas invertidas en cada tarea, responsable de la tarea, fecha de inicio/final y el estado en el que se encuentra esta tarea.
